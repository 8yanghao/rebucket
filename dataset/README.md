# Dataset format
数据使用json格式存储。
格式如下:
```json
[{stack_id:0,
    stack_arr:[{
        symbol:"main",
        file:"main.java",
        line:100},
        {
        symbol:"Sort.quick_sort",
        file:"Sort.java",
        line:123},
        {
        symbol:"Sort.print_result",
        file:"Sort.java"
        line:111},
        duplicated_stack:-1
        ]
    },
    {stack_id:1,
    stack_arr:[{
        symbol:"main",
        file:"main.java",
        line:100},
        {
        symbol:"Sort.quick_sort",
        file:"Sort.java",
        line:123},
        {
        symbol:"Sort.print_result",
        file:"Sort.java"
        line:111},
        duplicated_stack:-1
        ]
    },
    {...}
]
```
